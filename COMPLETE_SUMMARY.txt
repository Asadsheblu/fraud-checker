================================================================================
                    FRAUDSHIELD SaaS - সম্পূর্ণ প্রজেক্ট সারসংক্ষেপ
================================================================================

এই ডকুমেন্ট সবকিছুর একটি সাধারণ ওভারভিউ। প্রতিটি বিস্তারিত জানার জন্য সম্পূর্ণ 
ডকুমেন্টেশন পড়ুন।

================================================================================
১. এটা কি?
================================================================================

FraudShield একটি জালিয়াতি সনাক্তকরণ সফটওয়্যার যা:

✓ Shopify দোকানে জালিয়াতি ধরে
✓ WordPress (WooCommerce) এ কাজ করে
✓ নিজের API দিয়ে যেকোনো সাইটে ব্যবহার করা যায়
✓ প্রতিটি অর্ডার স্বয়ংক্রিয়ভাবে বিশ্লেষণ করে
✓ স্কোর দেয় (0-100) এবং সুপারিশ করে

================================================================================
२. কিভাবে কাজ করে?
================================================================================

সাধারণ ফ্লো:

  গ্রাহক → অর্ডার দেয় → FraudShield এ যায় → ৭টি নিয়ম দিয়ে চেক হয় 
  → স্কোর দেওয়া হয় (০-১०००) → রেজাল্ট বলা হয়:
  
    ✅ নিরাপদ (0-30)      → Approve করুন
    ⚠️ ঝুঁকিপূর্ণ (31-60) → রিভিউ করুন
    ❌ জালিয়াতি (61-100) → ব্লক করুন

৭টি চেক:
  1. বড় অর্ডার?          (0-30 পয়েন্ট)
  2. ঠিকানা মিলছে না?    (0-20 পয়েন্ট)
  3. বিপজ্জনক দেশ?      (0-25 পয়েন্ট)
  4. খুব তাড়াতাড়ি অর্ডার? (0-15 পয়েন্ট)
  5. ব্ল্যাকলিস্টে আছে?   (0-30 পয়েন্ট)
  6. অজানা ডিভাইস?      (0-20 পয়েন্ট)
  7. সন্দেহজনক ইমেইল?    (0-10 পয়েন্ট)

================================================================================
३. কিভাবে শুরু করবেন?
================================================================================

EASY QUICK START (৫ মিনিট):

Step 1: সিস্টেম চালু করুন
  ➜ npm run dev                    (ফ্রন্টেন্ড, পোর্ট: 8085)
  ➜ cd server && npm run dev       (ব্যাকএন্ড, পোর্ট: 3001)

Step 2: ব্রাউজার খুলুন
  ➜ http://localhost:8085

Step 3: লগইন করুন
  ইমেইল: test@example.com
  পাসওয়ার্ড: password123

Step 4: API Key তৈরি করুন
  ড্যাশবোর্ড → API Keys → Create New Key

Step 5: ব্যবহার করুন
  Shopify: Integrations → Connect Shopify (2 ক্লিক)
  WordPress: Plugin ইনস্টল করুন + API Key দিন
  নিজের API: API রেফারেন্স পড়ুন

४. দেখবেন কি?
================================================================================

ড্যাশবোর্ড:
  • Total Orders: সব অর্ডার সংখ্যা
  • Safe/Risky/Fraud: প্রতিটি ক্যাটেগরি
  • Revenue Protected: বাঁচানো টাকা
  • Trends: গত ৩০ দিনের গ্রাফ

Orders পাতা:
  • সব অর্ডার তালিকা
  • প্রতিটির স্কোর এবং কারণ
  • ম্যানুয়ালি পরিবর্তনের অপশন

Settings:
  • API Keys ম্যানেজমেন্ট
  • Integrations সংযোগ
  • Webhooks সেটআপ
  • প্ল্যান পরিবর্তন

Admin Panel (শুধু এডমিনদের):
  • সব ইউজার দেখুন
  • জালিয়াতির নিয়ম সম্পাদনা
  • সিস্টেম স্ট্যাটিস্টিক্স
  • Audit logs (সব কার্যকলাপ)

================================================================================
५. প্রযুক্তি কি কি ব্যবহার হয়েছে?
================================================================================

Frontend (ফ্রন্টএন্ড):
  • React 18 (ব্যবহারকারী ইন্টারফেস)
  • TypeScript (নিরাপদ কোড)
  • Tailwind CSS (সুন্দর ডিজাইন)
  • SWR (দ্রুত ডেটা লোডিং)

Backend (ব্যাকএন্ড):
  • Express.js (Node.js ওয়েব সার্ভার)
  • TypeScript (নিরাপদ কোড)
  • JWT (নিরাপদ লগইন)
  • Stripe (পেমেন্ট)

Database (ডাটাবেস):
  • Supabase (PostgreSQL)
  • 13টি টেবিল
  • Row Level Security (RLS)
  • Daily backup

Integrations:
  • Shopify API
  • WordPress WooCommerce
  • Resend (ইমেইল)
