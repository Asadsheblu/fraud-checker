┌──────────────────────────────────────────────────────────────────────────────┐
│                      🔄 Fraud Detection Process                              │
└──────────────────────────────────────────────────────────────────────────────┘

গ্রাহক অর্ডার দেয়
    ↓
FraudShield API এ যায়
    ↓
७টি Rule দিয়ে Score দেওয়া হয়:
    ├─ Amount Check (অর্ডার পরিমাণ)        → 0-30 points
    ├─ Address Mismatch (ঠিকানা মিল)      → 0-20 points
    ├─ Country Risk (দেশ ঝুঁকি)           → 0-25 points
    ├─ Velocity Check (দ্রুত অর্ডার)       → 0-15 points
    ├─ Blacklist Match (ব্ল্যাকলিস্ট)      → 0-30 points
    ├─ Device Fingerprint (ডিভাইস ট্র্যাক)  → 0-20 points
    └─ Email Domain (ইমেইল যাচাই)        → 0-10 points
    ↓
Total Score (0-100) দেওয়া হয়
    ↓
Risk Level নির্ধারণ:
    ├─ 0-30:  ✅ Safe     → অনুমোদন করুন
    ├─ 31-60: ⚠️ Risky    → রিভিউ করুন
    └─ 61-100: ❌ Fraud   → ব্লক করুন
    ↓
Result পাঠানো হয় গ্রাহক/স্টোরের কাছে
