# FraudShield SaaS - সম্পূর্ণ বাংলা গাইড

## এটা কি?

FraudShield একটি **জালিয়াতি সনাক্তকরণ প্ল্যাটফর্ম** যা অনলাইন স্টোরে জালিয়াতি ঠেকায়।

### কীভাবে কাজ করে?

```
গ্রাহক → অর্ডার দেয় → FraudShield → ৭টি নিয়ম দিয়ে চেক → রিজাল্ট
                            ↓
                     জালিয়াতির স্কোর (০-১০০)
                            ↓
                 নিরাপদ / ঝুঁকিপূর্ণ / জালিয়াতি
```

## ৭টি জালিয়াতি সনাক্তকরণ নিয়ম

| নিয়ম | কি দেখে | স্কোর |
|------|---------|-------|
| **অর্ডার পরিমাণ** | অস্বাভাবিক বড় টাকার অর্ডার | ০-৩০ |
| **ঠিকানা মিল** | শিপিং = বিলিং ঠিকানা | ০-২০ |
| **দেশ ঝুঁকি** | উচ্চ ঝুঁকির দেশ থেকে | ০-২৫ |
| **দ্রুত অর্ডার** | একই ব্যক্তি দ্রুত অনেক অর্ডার | ০-১৫ |
| **ব্ল্যাকলিস্ট** | আগে কখনো জালিয়াতি করেছে? | ০-३० |
| **ডিভাইস ট্র্যাক** | একই ডিভাইস থেকে জালিয়াতি? | ०-२० |
| **ইমেইল চেক** | ইমেইল ডোমেইন বিশ্বস্ত? | ०-१० |

## স্কোর মানে কি?

```
স্কোর ০-३०  → ✅ নিরাপদ (Approve করুন)
স্কোর ३१-६०  → ⚠️ ঝুঁকিপূর্ণ (রিভিউ করুন)
স্কোর ६१-१०० → ❌ জালিয়াতি (Block করুন)
```

### প্রথম পদক্ষেপ: অ্যাকাউন্ট তৈরি করুন

```
১. যান: https://fraudshield.io
२. "Sign Up" ক্লিক করুন
३. ইমেইল এবং পাসওয়ার্ড দিন
४. ইমেইল যাচাই করুন
५. ড্যাশবোর্ডে লগইন করুন
```

### দ্বিতীয় পদক্ষেপ: API Key তৈরি করুন

```
ড্যাশবোর্ড → বাম দিকে "API Keys" → "Create New Key"

নাম দিন: যেমন "My Shopify" বা "My Website"

এটা পাবেন:
sk_live_abc123xyz...

গুরুত্বপূর্ণ: এটা সেভ করুন! পরে দেখতে পারবেন না।
```

### তৃতীয় পদক্ষেপ: Integrate করুন

#### Shopify এর সাথে

```
Dashboard → Integrations → "Connect Shopify"

Shopify দেবে authorization দেওয়ার জন্য অনুমতি।
দিন "Install" ক্লিক করুন।

FraudShield স্বয়ংক্রিয়ভাবে:
✓ Webhook সেট করবে
✓ প্রতিটি নতুন অর্ডার চেক করবে
✓ জালিয়াতি ঠেকাবে
```

#### WordPress (WooCommerce) এর সাথে

```
WordPress Admin Panel তে যান

Plugins → Add New → Upload Plugin

fraudshield-woocommerce.php ফাইল আপলোড করুন

"Activate Plugin" ক্লিক করুন

Plugin Settings এ যান:
- API Key পেস্ট করুন
- সেভ করুন
- শেষ!

এখন প্রতিটি order checkout এ চেক হবে।
```

#### নিজের API দিয়ে

```bash

    "deviceId": "device_12345",
    "orderId": "ORD-001"
  }'

Response পাবেন:
{
  "orderId": "ORD-001",
  "riskScore": 25,
  "riskLevel": "safe",
  "triggers": [],
  "recommendation": "APPROVE",
  "confidence": 0.98
}
```

## প্রশ্ন ও উত্তর

### Q: আমার API Key হারিয়ে গেছে। কি করব?
A: Dashboard → API Keys → সেই Key এর পাশে "Regenerate" বাটনে ক্লিক করুন। নতুন key পাবেন।

### Q: কত খরচ পড়বে?
A: ৩টি প্ল্যান আছে:
```
FREE: ১০০ অর্ডার/মাস = ফ্রি
PRO: ১০,০০০ অর্ডার/মাস = ৫০০ টাকা
ENTERPRISE: আনলিমিটেড = কাস্টম মূল্য
```

### Q: কোন অর্ডার ব্লক হলে গ্রাহক জানবে?
A: হ্যাঁ, স্বয়ংক্রিয় ইমেইল পাঠানো হয়।
```
বিষয়: "আপনার অর্ডার যাচাইয়ের অপেক্ষায়"

বার্তা:
আপনার নিরাপত্তার জন্য, আমরা আপনার অর্ডার যাচাই করছি।
কয়েক ঘন্টার মধ্যে আপডেট পাবেন।

সাপোর্ট: support@fraudshield.io
```

### Q: ম্যানুয়ালি ব্লক করা অর্ডার কি আবার আনব্লক করতে পারি?
